=== Switcher i18n

Header multilingue standardisé dans chaque fichier. La langue courante
est en gras (`**…​**`), les autres sont des liens relatifs `README.<code>.md`.

Exemple (français) :
```
🌐 Langues : [English](README.md) | **Français**
```

Versions non-françaises affichent le switcher complet à 10 entrées :
```
🌐 Languages: [English](README.md) | [中文](README.zh.md) | [हिन्दी](README.hi.md)
            | [Español](README.es.md) | [Français](README.fr.md) | [العربية](README.ar.md)
            | [বাংলা](README.bn.md) | [Português](README.pt.md) | [Русский](README.ru.md)
            | [اردو](README.ur.md)
```

=== Support RTL (Arabe, Urdu)

README.ar.md et README.ur.md (dans les deux répertoires) :
- contenu en script natif (Arabic / Nastaliq)
- switcher i18n ordonné (العربية et اردو en gras sur leur fichier)
- badges/markers alignés sur le master EN
- Aucune directive `dir="rtl"` HTML ajoutée — GitHub Plugin Portal et
  l'agent de rendu markdown détectent automatiquement le script.
